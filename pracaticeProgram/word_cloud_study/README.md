
python词云
==========

本词云练习是按照[官方库](https://github.com/amueller/word_cloud)案例做的，基本上就是了解熟悉一下整个库的使用方式方法。

### 安装

#### pip安装

```` 
pip install wordcloud
````

#### whl文件安装

下载[.whl文件](http://www.lfd.uci.edu/~gohlke/pythonlibs/#wordcloud)，使用pip install 安装,会自动安装依赖
